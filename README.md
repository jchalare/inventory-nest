## Installation

```bash
$ yarn install
```

## Environment

```bash
# clone the '.env.template' file and rename to '.env' and change the environment variables 
```

## Inventory Nest DB

```bash
docker-compose up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
## Populate DB

```bash
 localhost:3000/api/seed
```
